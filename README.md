# Douban_spider

爬取豆瓣**电影**(**不包含** 电视剧 综艺 动漫 纪录片 短片 )

[TOC]

## Installation

```bash
$ git clone https://github.com/1-riverfish/douban_spider.git
$ pip3 install requests progressbar2
```

> **python3  pip3 is pre-installed.**

## Usage

```bash
$ cd douban_spider
$ chmod +x *.py
$ ./douban_movie_subjectid_spider.py
```
